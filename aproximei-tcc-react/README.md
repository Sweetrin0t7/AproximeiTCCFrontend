# AproxiMEI Frontend

## Informações do Projeto

AproxiMEI é uma plataforma para conexão eficiente entre consumidores e microempreendedores da região carbonífera do RS.  
Esta é a parte **frontend**, desenvolvida com React, TypeScript e Tailwind CSS, utilizando o design system `shadcn-ui`.

## Tecnologias Utilizadas

- React + TypeScript
- Vite
- Tailwind CSS
- shadcn-ui
- React Router Dom
- React Query

## Como rodar o projeto localmente

### Pré-requisitos

- Node.js 
- npm

### Passos

```bash
# 1. Clone o repositório
git clone <URL_DO_SEU_REPOSITORIO>

# 2. Entre na pasta do projeto
cd aproximei-tcc-react

# 3. Instale as dependências
npm install

# 4. Inicie o servidor de desenvolvimento
npm run dev
